2. Weekly graph question¶

Below are the graphs from last week's weekly graph question. Suppose you wanted to cause the viewer to examine the last three months of data first. How could you use preattentive attributes to do that? Suggest several possibilities.

One attribute to use is leveraging color to draw attention. By using a color like blue for the last 3 months of the line, and grey for the rest, it will draw the users eyes to the period of analysis.

Another option is to add data labels to the plot for the last 3 months. This causes the same effect, the user will focus on the point of the plot which contains the data labels.

Size can also be leveraged. If we add data labels, we can increase the size of the labels to draw attention.
